# Airline eTicket Booking System (ATBS)

## Overview

This project is a database system developed to manage airline operations such as flights, customers, bookings, seat allocation, and payments. It aims to reduce errors and improve data organization using database management system concepts.

## Problem Statement

Manual airline management systems are inefficient and prone to errors. This project provides a structured database solution to manage operations accurately and efficiently.

## Objectives

* Create a database using SQL
* Perform CRUD operations
* Maintain data integrity using keys and constraints
* Normalize the database up to Third Normal Form

## Features

* Customer management
* Flight and airline management
* Seat tracking
* Booking management
* Payment record management
